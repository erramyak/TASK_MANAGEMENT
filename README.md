# Task Management Application

A full-stack Task Management Web Application that allows users to securely manage their daily tasks. The application provides user authentication, task creation, editing, updating, deletion, and status tracking through a responsive and user-friendly interface.

## Features

- User Registration and Login
- JWT Authentication & Authorization
- Password Encryption using bcrypt.js
- Create, Read, Update and Delete (CRUD) Tasks
- Task Status Management (Pending, In Progress, Completed)
- Responsive User Interface
- RESTful API Integration
- MySQL Database Connectivity
- Secure Protected Routes

## Tech Stack

### Frontend

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Authentication
- JSON Web Token (JWT)
- bcrypt.js
